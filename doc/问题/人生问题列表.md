# 如何有逻辑?
事情有两面，看两面性，好坏进行比较。好处远大于坏处时，可选择。
进行横向比较，而不是自己跟自己对比
忽略个案，讲数据
避免悲观，避免小题大作，避免宁可信其有，不可信其无，讲数据，讲概念。
避免一厢情愿，祈祷。勇于自己内心认错，不重复犯相同的错误（不二过，亡羊补牢）
追根究底的能力，看透事特本质的能力
多维度考察事情，不是单一维度得出结论
