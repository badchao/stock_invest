# 投资相关人性的弱点:
自私、逐利、短视、贪婪、恐惧、盲从、迷信权威，寻找优越感。与人要有利益交换，让人知道你的价值
逐利：都要有利益交往
短视：做事情马上要看到效果，容易极端主义（为了短期目标，不计后果）
盲从：从众，跟风，为了融入集体，为了安全感，为了不承担独立决策压力，导致跟风。人要有独立思考能力
恐惧：要研究历史，经常发生，克服恐惧
迷信权威：多读书，认可自己，你比他们历害
健忘：1根阳线改三观，不行再来一根，人马上变乐观，忘了伤痛
集权：对权力的渴望（中国政治制度根源）
