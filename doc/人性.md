# 人性伦理
叔本华认为人类行为的动机可以分成三种：希望自己快乐，希望他人痛苦，希望别人快乐。
他将这三种动机分别概括为利己、恶毒、同情，其中利己和恶毒是非道德的推动力，只有基于同情是真正的道德行为。

利己
恶毒
同情

由于一切行为的动机都出于利己、恶毒、同情这三者，所以一个人的道德程度就可以看成这三者在他的性格中的比例。
同情在这三者中所占比例越大，则一个人的道德程度越高。
